# Rural Brain Drain Analysis

## Overview

This project investigates rural population decline across the United States using USDA Rural-Urban Continuum Codes, Census American Community Survey (ACS) data, and Bureau of Labor Statistics (BLS) employment data.

The goal is to identify counties experiencing significant outmigration and explore how educational attainment and employment opportunities influence population loss.

## Research Question

Why are some rural counties losing population faster than others?
